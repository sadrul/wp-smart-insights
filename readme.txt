== Description ==

Smart Insights is a powerful, privacy-first WordPress plugin that provides comprehensive content intelligence and user experience optimization tools. It combines AI-powered content analysis, visual heatmaps, SEO scoring, and user journey tracking to help you create better content and improve user engagement.

= Key Features =

* **AI-Powered Content Analysis**
  * Readability scoring and reading level assessment
  * Sentiment analysis and emotional tone detection
  * Writing style and tone consistency analysis
  * Keyword density and content optimization suggestions
  * Repetition detection and improvement recommendations

* **Visual Heatmaps**
  * Click tracking with visual overlay
  * Scroll depth analysis
  * Hover pattern tracking
  * Real-time data collection
  * Canvas-based heatmap visualization
  * UX issue detection and warnings

* **SEO Intelligence**
  * Comprehensive SEO scoring system
  * Heading structure analysis
  * Meta tag optimization
  * Internal linking suggestions
  * Image alt tag verification
  * One-click SEO fixes

* **User Journey Playback**
  * Anonymous session recording
  * Step-by-step session replay
  * Interaction logging and analysis
  * Behavior pattern identification
  * Privacy-first design

* **Advanced Analytics**
  * Real-time analytics dashboard
  * Export capabilities (CSV, JSON, XML, Excel)
  * Email notifications and reports
  * REST API for external integrations
  * Webhook support for automation

* **Privacy & Compliance**
  * GDPR compliant by design
  * Built-in consent management
  * Data anonymization
  * No third-party tracking
  * Full data export/deletion control

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/smart-insights` directory, or install the plugin through the WordPress admin screen using the Add New option.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Smart Insights menu in the admin panel to configure the plugin.
4. Add your OpenAI API key for enhanced AI analysis (optional).

== Frequently Asked Questions ==

= Is this plugin GDPR compliant? =

Yes, Smart Insights is designed with GDPR compliance in mind. All data is anonymized, users must provide consent before tracking, and you have full control over data export and deletion.

= Do I need an OpenAI API key? =

No, the plugin works without an OpenAI API key, but adding one enables advanced AI-powered content analysis features.

= How much does the AI analysis cost? =

AI analysis costs depend on your OpenAI usage. Typical content analysis costs around $0.01-0.05 per analysis. You can monitor usage in the AI Analysis section.

= Can I export my data? =

Yes, Smart Insights provides comprehensive export capabilities in multiple formats (CSV, JSON, XML, Excel) for all collected data.

= Does this work with my theme? =

Smart Insights is designed to work with all WordPress themes. The tracking is lightweight and shouldn't affect your site's performance.

= Is my data secure? =

Yes, all data is stored on your server and never shared with third parties. The plugin uses WordPress security best practices.
